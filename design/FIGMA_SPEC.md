# City Jam — Figma file structure

Create a Figma file with these pages. Token values live in `design/tokens.json`.

## Foundations
- Color modes: Dark (default), Light — bind to Figma Variables from tokens.json
- Typography: Bebas Neue / Inter / Special Elite
- Grain overlay: noise tile 110px dark, 140px light
- Radii: vinyl-full, vinyl-card, pill, chip
- Elevation: groove inset + disc shadow

## Components
- VinylDisc, VinylCard, CassetteChip, WaveformBadge, MicHero
- GrainOverlay, ThemeToggle
- OAuthButton (Google, Facebook, Apple)
- CoverArtFrame variants: 4:3, 3:4, square

## Patterns
- Homepage hero + ambient map
- Scene AudioPostCard (vinyl frame)
- Artist feature profile (cover art hero)
- Blind Echo scanning state
- Auth card (vinyl frame)
- Nav rooms map (5 tabs + tools)

## Carousel
- MusicianGallery: vinyl-thumbnail slides, gold groove, center-focus scale on snap
